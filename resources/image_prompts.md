## Image Generation Recommendations

### Recommended AI Tools
1. **Midjourney** - Best for photorealistic moving/service imagery
2. **DALL-E 3** - Good for infographic/package designs
3. **Stable Diffusion XL** - Good alternative with more control

### Logo Seeding Instructions
For images requiring the company logo:
1. Generate the base image first
2. Use image editing software (Photoshop, Canva, Figma) to overlay the logo
3. Alternatively, use AI inpainting to add logo to truck sides/uniforms
4. Ensure logo is visible but not overpowering

### Logo File Location
The company logo should be available at: `/public/images/logo.png`
Reference the WordPress export for the original logo: `184101456_padded_logo.png`
